# 🏔️ Mountain Website - Landing Page Sederhana

Selamat datang di repositori **Mountain Website**! 🌲 

Proyek ini adalah sebuah halaman utama (landing page) sederhana namun menarik yang didedikasikan untuk menampilkan keindahan alam pegunungan. Proyek ini sangat cocok digunakan sebagai referensi bagi pecinta alam, pendaki, atau siapa saja yang ingin belajar membuat antarmuka web dasar dengan nuansa alam.

## ✨ Fitur

- **Desain Minimalis & Bersih**: Tampilan yang rapi agar mata fokus pada keindahan visual pegunungan.
- **Sangat Sederhana**: Dibuat dengan struktur kode yang simpel dan mudah dipahami, sangat ramah bagi pemula.
- **Ringan & Cepat**: Karena tidak menggunakan framework yang berat, website ini dapat dimuat dengan sangat cepat.

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun murni menggunakan teknologi web dasar:
- **HTML5** untuk kerangka dasar dan struktur halaman.
- **CSS3** untuk desain, tata letak, dan mempercantik tampilan.
- **Vanilla JavaScript** untuk memberikan interaktivitas dasar.

## 🚀 Cara Menjalankan di Perangkat Anda

Karena ini adalah proyek website statis, Anda tidak perlu melakukan instalasi sistem (seperti Node.js) atau melakukan konfigurasi yang rumit.

1. **Unduh Proyek**:
   Clone repositori ini menggunakan terminal:
   ```bash
   git clone https://github.com/username/mountain-website.git
   ```
   *Atau Anda bisa langsung mengunduh file ZIP dan mengekstraknya.*

2. **Buka folder proyek**:
   Buka folder `mountain-website` yang telah diunduh di komputer Anda.

3. **Jalankan Website**:
   Cukup klik dua kali pada file `index.html`. Halaman tersebut akan otomatis terbuka di browser bawaan Anda (Google Chrome, Mozilla Firefox, dll).

## 📂 Struktur Folder

```text
mountain-website/
├── index.html        # File utama yang berisi struktur halaman
├── README.md         # Dokumentasi proyek (file ini)
├── image/            # Folder untuk menyimpan semua aset gambar pegunungan
└── dist/
    ├── css/
    │   └── style.css # File untuk mengatur gaya (warna, font, tata letak)
    └── js/
        └── script.js # File skrip untuk animasi atau fungsi interaktif
```
